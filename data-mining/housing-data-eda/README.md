# Housing Dataset - Exploratory Data Analysis (EDA)

## Project Description

This project performs **Exploratory Data Analysis (EDA)** on the California Housing dataset.

The goal is to understand relationships between geographic location, population, income, and housing prices using visualization and statistical analysis.


## Dataset

The dataset includes features such as:

- longitude
- latitude
- population
- median_income
- median_house_value


## Methodology

The analysis includes:

### 1. Data Splitting
- Train-test split (80/20)

### 2. Visualization
- Geographic distribution of houses
- Population-based scatter plot
- House value color mapping

### 3. Feature Analysis
- Correlation matrix
- Feature ranking based on correlation with target

### 4. Feature Relationships
- Scatter matrix for top correlated features
- Income vs house value relationship


## Key Insights

- Higher income strongly correlates with higher house prices
- Population affects visualization density
- Geographic location plays an important role in housing price distribution


## Visualizations

- Longitude vs Latitude map
- Population-scaled scatter plot
- Colored house value heat map
- Correlation matrix
- Scatter matrix of top features
