
# Prime Divisor Beauty Checker

## Problem

A number is considered beautiful if:

- The count of its prime divisors
- Is itself one of the prime divisors of the number

The program:

1. Checks all odd numbers from 1 to n
2. Finds beautiful numbers
3. Computes their sum
4. Prints the reversed sum

If no valid number exists:

```text
NOT FOUND!
```


## Example

For number 30:

Prime divisors:

```text
[2, 3, 5]
```

Count:

```text
3
```

Since 3 is inside the divisor list, the number satisfies the condition.


## Concepts Used

- Prime number checking
- Divisor analysis
- Nested loops
- List processing
- Number reversal


## Complexity

Brute-force implementation:

O(n²)
